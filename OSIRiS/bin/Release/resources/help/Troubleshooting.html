Troubleshooting

Technical Overview ››

The formatter produces blank/raw disks.


Choose a disk that supports the file system you want to use.

For a run down of supported disks and file systems see here,

alternatively hovering over the file system buttons in OSIRiS will

provide a brief description.



The formatter appears to format the disk but
the file system displayed is the same.


The operating system may have a process accessing the disk or

the disk may have been write locked. You can format the disk

into an alternative file system and confirm that succeeds, then try

your desired file system.



The cancel auto-shutdown link on the desktop
does nothing.


The cancel auto-shutdown link is intended for use in such times that the

machine is in use and the auto-shutdown time rolls around. It is not intended

to postpone auto-shutdown for that day. If you are using a machine and it

begins to shutdown, press cancel/close and then double click the icon.

The machine will need to be manually shutdown when you are done.
