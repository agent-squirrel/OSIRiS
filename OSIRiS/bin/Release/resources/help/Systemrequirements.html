System requirements

Technical Overview ››

OSIRiS system requirements are as follows:


  • .NET 4.5 (Included in Windows 8/8.1 and 10.
  • Keyboard and Mouse/Trackpad.
  • A multi-core processor or hyper threading. (OSIRiS will run on a single core
    machine, but will be significantly slower)
