User Interface Introduction ››

Choose the Sell tab from the top of OSIRiS.


The user interface for selling a display/demo computer

looks like FIG A.

Click Quit at any time to stop using OSIRiS.


FIG A


Most of the controls are labelled with relatively straight forward
overviews of what they do. If you mouse over any of the controls

or labels, a tool tip will provide a brief description of it's intended

function.


To begin choose a completion behaviour from FIG B. This is

the action that will be performed when OSIRiS has finished running.

Next input the customer's desired username, this can be left as

'User' if desired, the new user account will have no password.


FIG B


Choose whether to display the console output, which will

print messages to the screen as OSIRiS performs tasks.
Then press Run. FIG C.
