Usage Scenarios

Getting Started ››

Here are a collection of scenarios that you may encounter as a team member

and how to tackle them with OSIRiS.


Customer in a hurry.


You have been asked by a customer if you can pack up a display computer for

sale to them. They have stated they are in a hurry and have asked how long it

will take to prep the machine.

In this scenario, the answer is somewhere in the region of 5 - 10 minutes.
OSIRiS only takes seconds to execute and the longest part of the sell

process is the time the machine takes to ready it's new user account once

it has rebooted. Also boxing the machine up can often take more time than

OSIRiS requires to run.

If the customer is happy to wait a little while the machine gets going when they

arrive home. You can elect to choose the shutdown option when selling the

machine, as this will remove all of the Officeworks information, delete the user

accounts and create a new one for the customer. However instead of rebooting

into the new account (often the longest part) the machine will power off. When the

customer arrives home, the machine will power on into the new account and run

through it's first time setup routine.



Customer wants to use an external hard disk drive

with their game console.


You have been asked by a customer to format an external hard drive for use

with the customers game console.


This question depends on some more information, we need to find out what

type of game console the customer has first off.

PS3/XBOX 360: - FAT32 would be the ideal file system as these consoles

cannot use exFAT. The XBOX 360 can use NTFS but without knowing the

system software version we can not be certain if it will work.


PS4/XBOX ONE: - exFAT would be the best file system for the newer consoles.
They can both read and write to this file system and it also provides great

compatibility with the customers computer too.


Next up, explain to the customer that the formatting process will take a maximum

of 5 minutes. Depending on the drive size and file system this can vary wildly, from

seconds with 1- 2TB drives and exFAT all the way up to 5 minutes for FAT32 on a 2TB

volume. Your mileage may vary, however it is never a particularly long time.



Customer uses Windows only and wants a drive for

storing photos and memories.


Most shop bought external hard drives are already formatted NTFS. On the rare

occasion that a customer brings in an already reformatted drive and wants to

use it for long term storage in a Windows only environment, NTFS provides

some level of fault tolerance and data corruption prevention. It is however not a

replacement for a good backup.



Customer has a Mac and a Windows PC.


A customer wants to use their drive on a PC and a Mac.

exFAT is the best option for this scenario as it doesn't come with the inherent limitations

of FAT32 but can still be read and written to by both Windows and OS X. See here.
